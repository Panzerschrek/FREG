    • freg

    Game for Linux terminals (and maybe for something else)

    Project maintained by mmaulwurff Hosted on GitHub Pages — Theme by mattgraham

    FREG is a free-roaming elementary game. Inspired by Minecraft and Angband. This is a version of FREG with text graphics.

    Development stage: implementing basics.

    Language: C++, Qt

    Non-standard libraries: libncurses5

    Compile: qmake freg-nox.pro; make;

    (do not forget include -lncurses in Makefile)

    Or open freg-nox with QtCreator.

    Development is run on GNU/Linux, but on MS Windows should work too with PDCurses.

    Documentation

    Now only singleplayer is developed, multiplayer is planned.
